## Rock, Paper, Scissors, Lizard, Spock!
#### [Trabajo Final Integrador](https://github.com/unq-ui/material/blob/master/TFIs/2020s2-TFI.md#uis--trabajo-final-integrador-tfi--2c-2020) - Construcción de Interfaces de Usuario de UNQ - Ailín Sofía Patiño

## Cómo jugar:

1. Abrir la consola en la carpeta que desea instalar el juego y ejecutar el siguiente comando:
```
git clone https://github.com/ailinsp/unq-ui-ailin-patino-trabajo-final.git
```

2. Ingresar a la carpeta del proyecto:
```
cd unq-ui-ailin-patino-trabajo-final
```

3. Instalar las dependencias:
```
npm install
```

4. Iniciar el juego:
```
npm start
```
